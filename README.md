# MERN Stack CRUD Application

This application is built using the MERN stack: 
- MongoDB
- Express.js
- React (use Vite.js -> React.js)
- Node.js

# Backend setup
- npm init -y
- npm install express mongoose dotenv
- npm i nodemon

# Frontend setup
- cd frontend
- npm create vite@latest .
  //choose React and Javascript version
